## 3. Diskussionsfragen

### Schmerzpunkte heute
- Welchen Kontext gebt ihr eurem AI-Assistenten am häufigsten mit?
- Welche Informationen müsst ihr heute am mühsamsten zusammensuchen?

### Struktur & Priorisierung
- Gibt es Sektionen, die **alle** im Marketing-Team als "Primär" haben sollten? (z.B. eine "Ryzon Essentials"-Sektion mit den absoluten Grundlagen)
- Welche Sektionen sind **rollenspezifisch** und welche sind **teamweit** relevant?

### Aktualität & Quellen
- Wie oft ändert sich euer Kontext wirklich? (Zielgruppen = stabil über Monate, Performance = wöchentlich, Events = laufend)
- Welche Datenquellen fehlen in der Liste? (Slack-Threads, Kunden-Reviews, Shopify-Daten, BI-Reports...)

### Flexibilität
- Braucht ihr die Möglichkeit, eine Sektion **temporär hochzustufen**? (z.B. während eines Produkt-Launches die Produktlinien-Sektion auf "Primär" setzen)
- Gibt es Situationen, in denen ihr **das Profil eines Kollegen** nutzen wollen würdet? (z.B. als Performance Marketer kurz die Creative-Strategist-Brille aufsetzen)

### Erlebnis
- Sollte der Agent am Anfang einer Session sagen, welchen Kontext er geladen hat — oder soll das unsichtbar passieren?
- In welcher Sprache wollt ihr mit dem Agent kommunizieren? (Der Kontext ist auf Englisch gespeichert, der Agent antwortet in eurer Sprache)

## 4. Offene Design-Fragen

Diese Fragen helfen uns, die nächste Phase zu planen:

| Frage | Optionen | Eure Meinung |
|-------|---------|-------------|
| "Ryzon Essentials"-Sektion für alle? | Ja, Grundlagen für jeden / Nein, alles rollenspezifisch | |
| Profil-Wechsel erlauben? | Fest pro Person / Temporäre Overrides möglich | |
| Kontext-Transparenz? | Agent zeigt geladenen Kontext / Unsichtbar | |
| Wer pflegt manuelle Sektionen? | Jeder selbst / Ein Verantwortlicher pro Sektion / Brand Team zentral | |
| Feedback-Loop? | Agent fragt "War diese Info hilfreich?" / Passiv (wir messen Nutzung) | |
